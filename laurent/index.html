Programmation

More details on my programming page.
  • Phystricks est un module Python (en fait : Sage) pour faire des figures en LaTeX.
  • lora est mon programme de backup programmé en C++. Il aide également à ne pas oublier nos git commit quotidiens.
  • actors est un modèle d'acteurs programmé en Java. C'est le projet pour un cours de programmation concurrente à l'université de Padoue.
  • frtex est un package Java, utilisant le système d'acteur, qui permet des manipulations de fichiers LaTeX en java (en cours de développement). Pour l'instant la seule fonctionnalité est de substituer récursivement tous les \input par le contenu du fichier pointé.
